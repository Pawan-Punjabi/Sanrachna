Create a modern AI-powered web application that analyzes a floor plan image and suggests furniture products.

Core Functionality

The user can upload a floor plan image (PNG/JPG).

The system sends the image to a YOLO-based object detection model trained to detect furniture types such as:

bed

sofa

table

chair

wardrobe

sink

toilet

TV

dining table

The model returns bounding boxes and labels for detected furniture.

Display the annotated floor plan image with detected furniture highlighted.

For each detected furniture item, generate furniture suggestions from online stores like:

IKEA

Amazon

Show product cards with:

product image

price

rating

product link

store name

AI Integration

Backend should load a YOLOv8 or YOLOv11 model

Accept image input via API

Return detection results as JSON:

{
 "objects":[
  {"label":"sofa","confidence":0.92},
  {"label":"bed","confidence":0.88}
 ]
}
Product Recommendation Logic

For each detected object:

Search furniture APIs or product scraping using keywords like:

“modern sofa”

“queen size bed”

“wooden dining table”

Show 3–5 relevant products per item
